How to run
==========

Clone this repo and cd into it.

Then run the app, with any of these commands:
- nitric run
- nitric up -s <aws,azure,gcp>

If you deployed, then get the api url output from the above command:

```
$ export BASE_URL=<from above>
$ make test
```
